# Python Environment Checklist

- Python 3.11 yoki undan yangi versiyani o'rnating.
- Virtual environment yarating: `python -m venv .venv`
- Muhitni yoqing va `pip install pandas matplotlib openpyxl` o'rnating.
- Loyihada `data/`, `scripts/`, `reports/` papkalarini ajrating.
- Har bir dars uchun notebook emas, qayta ishlatiladigan script saqlang.

# Batch Processing Plan

1. Kiruvchi fayllarni `raw/` papkaga joylang.
2. Har bir fayl uchun format va schema tekshiruvini bajaring.
3. Xatoliklarni `exceptions.log` faylida jamlang.
4. Tozalangan natijani `processed/` papkaga yozing.
5. Yakuniy summary ni `reports/daily-summary.json` ko'rinishida saqlang.
